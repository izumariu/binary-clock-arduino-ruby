Binary clock with Arduino and Ruby

binary clock with arduino mega 2560

View project on GitHub

Für's EBK zum Spaß haben:

  1. Ruby installieren.
  2. Erweiterung für Ruby installieren: sudo apt-get install ruby-serialport
  3. Arduino anschließen und den Port von ihm herausfinden: CHRISTIAN FRAGEN!
  4. Ruby-Skript herunterladen.
  5. In den Ordner mit dem Skript navigieren und EXEKUTIEREN: ruby bclock.rb -d PORT -b 9600
  6. Have fun!
